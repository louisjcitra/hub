# Model Klasifikasi Gambar (MNIST)

## Overview

Model ini adalah model klasifikasi gambar sederhana yang dilatih menggunakan dataset MNIST (angka tulisan tangan).
Model ini dapat memprediksi angka dari 0 hingga 9.

## Training

Model dilatih menggunakan TensorFlow dengan arsitektur CNN sederhana.
Proses pelatihan dilakukan selama beberapa epoch untuk mencapai akurasi yang memadai.

## Usage

Berikut adalah contoh cara menggunakan model ini di Python:

```python
import tensorflow_hub as hub

# Load model dari TensorFlow Hub
model = hub.KerasLayer("[https://tfhub.dev/louisjcitra/klasifikasi-gambar/1](https://tfhub.dev/louisjcitra/klasifikasi-gambar/1)")
